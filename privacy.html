Privacy Policy

Last Updated: June 24, 2026

1. Information We Collect

We only collect foundational information strictly necessary to maintain environment security and verify user permissions. This includes:

  • Account Metrics: Your authenticated email address provided during system onboarding.
  • Profile Customization: Voluntarily uploaded profile images hosted securely via cloud cluster storage buckets.
  • Transaction Verification: Payment processing reference IDs (such as Razorpay transaction mappings) to validate membership status. We do not store raw card credentials or core payment authentication details directly.

2. How We Use Information

Your data parameters are evaluated solely to support core ecosystem functionality. Specific use cases include:

  • Authenticating secure dashboard initialization states.
  • Rendering personal visual profile elements across navigation sections.
  • Processing account level upgrades to clear authorization rules for protected categories.
  • Transmitting technical alerts or critical security channel broadcasts when configured.

3. Data Storage & Security Protocols

All database architectures are integrated with enterprise grade Firebase database systems and dynamic cloud security guards. Profile media elements are mapped to unique access chains based on active UIDs. Your information is strictly protected against unauthorized environment alterations or external distribution channels.

4. Third-Party Integrations

Creators Vault shares necessary tracking tokens exclusively with industry standard infrastructure modules to guarantee continuous system performance:

  • Firebase (Google LLC): To manage secure authentication session flows and persistent storage pipelines.
  • Razorpay: To process secure payment pathways safely without raw server exposure.

5. Cookies & Local Storage

Our platform leverages local system variables (`localStorage`) exclusively to memorize your custom layout parameters, such as interface darkness modes, screen dimming values, and standard toggle states, minimizing server load and ensuring smooth navigation.

6. User Control & Data Deletion

You preserve full governance over your digital identity vector. You may update or remove profile imagery links at any point through the operational setting drawer panel. For absolute profile deletion requests, reach out directly to operations.

7. Contact Support

If you have any policy inquiries or tracking concerns regarding our digital environment management, connect directly with our secure compliance team via support@creatorsvault.com.
